# Spark Lab Element PNG Pack

This pack contains one generated PNG tile for each of the 118 confirmed chemical elements.

## Contents

- `png_512/` — 118 individual PNG tiles, 512×512 px
- `svg_source/` — matching SVG source files for future resizing/editing
- `contact_sheet_118_elements.png` — preview sheet of all tiles
- `manifest.json` — metadata for all elements

## Design

Each tile includes:
- atomic number
- chemical symbol
- element name
- simplified element category

The artwork is generated for Spark Lab and is suitable as starter game assets for an educational children’s chemistry app.

## Licensing note

These images are generated assets, not copied photos or third-party artwork. Element names and symbols are factual chemistry data and were aligned with the IUPAC periodic table.
